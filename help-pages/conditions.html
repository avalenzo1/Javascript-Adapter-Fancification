Condition Statements

Home » Unit 1 » Condition Statements



if (condition) {


}

  • "if" should be all lowercase!
  • use braces {} as seen above.
  • Conditions compare two things
    • For greater than, use >
    • For less than, use <
    • For equal use, ==
    • For greater than or equal to, use >=
    • for less than or equal to, use <=
  • Usually, add 2 spaves in front of every line inside the braces.
